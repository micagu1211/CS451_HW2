The purpose of this application is to be able to type in any phrase from any language and it will translate the phrase into the Spanish language. 

This is just a beginning of the application. In the future I will update the application where it will be able to translate any phrase of any language into ANY language not just Spanish. 

For the application, I am using Python 2.7 and I am also using the Python package "goslate". "goslate" is a translating package in Python.
